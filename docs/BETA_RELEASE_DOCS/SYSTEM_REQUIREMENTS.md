# System Requirements - Cotton Candy Bot v2.1.0-beta

## Minimum Requirements
- **OS**: Windows 10/11 (64-bit) or Linux (Ubuntu 20.04+, Debian 11+)
- **CPU**: Dual-core 2.0GHz or higher
- **RAM**: 4GB
- **Disk Space**: 1GB available
- **Network**: Stable broadband internet connection
- **Runtime**: Node.js v18.0.0 or higher

## Recommended Specifications
- **CPU**: Quad-core 3.0GHz or higher
- **RAM**: 8GB+
- **Network**: Low latency connection to Solana RPC (QuickNode/Helius recommended)

## Software Prerequisites
- **Node.js**: [Download](https://nodejs.org/) (LTS version recommended)
- **Git**: (Optional, for updates) [Download](https://git-scm.com/)
- **Solana Wallet**: Hardware wallet or compatible software wallet for private key management.
